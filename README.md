# ZenviX — Official Website

Full MERN stack marketing website for ZenviX desktop app.

## Stack
- MongoDB · Express.js · React 18 · Node.js
- Tailwind CSS v3 + Framer Motion

## Setup

1. Install dependencies:
```bash
npm install
cd client && npm install
cd ../server && npm install
```

2. Configure environment:
```bash
cp .env.example .env
# Edit .env with your MongoDB URI and GitHub token
```

3. Run development:
```bash
npm run dev
```

Frontend: http://localhost:5173
Backend: http://localhost:5000

## Build

```bash
npm run build
```

## Production

```bash
npm start
```
